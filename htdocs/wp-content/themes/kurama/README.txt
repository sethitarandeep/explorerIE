Kurama
=========

Kurama is a very stylish theme built for all creative websites whether its about movies or a news blog or perhaps a portfolio of your work. This is stylish and fast theme with a very minimal design. Built on the Bootstrap 3 Framework, Kurama has brilliant Responsive performance on Mobile and Tablet Devices as well. 

Getting Started
---------------

How to Install the Theme?

1. Go to WP Admin > Appearance > Themes > Add New.
2. Search for "Kurama".
3. Click on Install and Activate.

Alternatively,

1. Go to WP Admin > Appearance > Themes > Add New > Upload.
2. Upload the kurama.zip file, in which you found this README.txt
3. Click on Install and Activate.

Setting it Up
-------------

This theme is built using Customizer. So to Configure theme settings like Featured Content Area, Sliders, Social Icons, etc from 
	
	Dashboard > Appearance > Customizer
